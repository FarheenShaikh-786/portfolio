# Farheen Shaikh — Portfolio

Premium React + Vite + Tailwind + Framer Motion portfolio.

## Setup

```bash
npm install
npm run dev
```

## Environment

Copy `.env.example` to `.env` and set `VITE_WEB3FORMS_ACCESS_KEY` (Web3Forms) for the contact form.

## Public assets

Add these to the `public/` folder:

| File | Purpose |
|------|---------|
| `profile.jpeg` | Hero profile photo |
| `resume.pdf` | Optional — “Download resume” button |

## Build

```bash
npm run build
npm run preview
```

Deploy the `dist` folder (e.g. Vercel: framework Vite, output `dist`).
